CanIsoNet: Cancer-specific Isoform Interaction Network

CanIsoNet serves as a gateway to understand the functional role of cancer-specific isoform switching events. We provide two main functionalities for the users:
1. Browsing various statistics on cancer-specific isoform switching events in each of the 1209 cancer samples from the Pan-Cancer Analysis of Whole Genomes (PCAWG) project
2. Exploring and visualising isoform-specific network disruptions in the STRING interaction network.

  • Download Interaction Disruption In Dominant Transcripts
    Processed data which includes all detected cMDTs, their sample information & disrupted interactions (comprising domain-domain interaction), string density score of genes.

  • Download STRING Interaction File
    Data includes STRING human protein-protein interaction data.

  • Download Isoform Specific Interaction File
    Data includes isoform specific interaction data where missed and remaining interactions are reported.

  • Download String Interaction Density Score
    Data includes STRING network interaction score for each gene. This file also includes STRING density score of genes which were not found as cMDT so that not reported in CanIsoNet.
