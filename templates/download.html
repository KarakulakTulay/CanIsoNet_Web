CanIsoNet: Disease-specific Isoform Interaction Network

CanIsoNet serves as a gateway to understand the functional role of disease-specific isoform switching events. We provide two main functionalities for the users:
1. Browsing various statistics on disease-specific isoform switching events
2. Exploring and visualising isoform-specific network disruptions in the STRING interaction network

  • Download Interaction Disruption In Dominant Transcripts in 27 Cancer Types
    Processed data which includes all detected dMDTs, their sample information & disrupted interactions (comprising domain-domain interaction), string density score of genes.

  • Download STRING Interaction File
    Data includes STRING human protein-protein interaction data.

  • Download Isoform Specific Interaction Network of Human Transcriptome
    Data includes isoform specific interaction data where missed and remaining interactions for human transcriptome.

  • Download String Interaction Density Score
    Data includes STRING network interaction score for each gene. This file also includes STRING density score of genes which were not found as cMDT so that not reported in CanIsoNet.

  • Download Isoform Specific Interaction Network of Mouse Transcriptome
    Data includes isoform specific interaction data where missed and remaining interactions for mouse transcriptome.

  • Download Interaction Disruption in Dominant Transcripts in OIH disease due to MDT Switch
    Processed data which includes all detected dMDTs, their sample information & disrupted interactions (comprising domain-domain interaction), string density score of genes.

  • Download Interaction Disruption in Dominant Transcripts in three diseases (Alzheimer's disease, Parkinson's disease, Behcet's disease) due to MDT Switch
    Processed data which includes all detected dMDTs, their sample information & disrupted interactions (comprising domain-domain interaction), string density score of genes.
